# Ingresos

Esta pantalla administra los ingresos del usuario autenticado.

## Qué permite hacer

- Crear filas nuevas.
- Editar categoría, descripción e ingreso.
- Filtrar registros por categoría.
- Eliminar registros marcados.
- Descargar un CSV con los datos.

## Campos

- Categoría.
- Descripción.
- Ingreso.

## Datos relacionados

- Se guarda en `ingresos.json` dentro de la carpeta del usuario.
- Se usa como base para el cálculo del balance general.
