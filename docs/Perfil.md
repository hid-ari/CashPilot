# Perfil

Esta pantalla centraliza la configuración de la cuenta activa.

## Qué permite hacer

- Ver el usuario autenticado.
- Cambiar el nombre mostrado del perfil.
- Cambiar la moneda.
- Cerrar sesión.
- Eliminar los datos del usuario.

## Datos relacionados

- La configuración se guarda en `settings.json` dentro de la carpeta del usuario.
- El cambio de perfil afecta solo a la cuenta activa.
