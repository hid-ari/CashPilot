# Gastos variables

Esta pantalla sirve para gestionar los gastos variables del usuario.

## Qué permite hacer

- Agregar nuevos registros.
- Editar categoría, descripción, presupuesto y valor actual.
- Filtrar por categoría.
- Eliminar filas seleccionadas.
- Exportar los datos en CSV.

## Campos

- Categoría.
- Descripción opcional.
- Presupuesto.
- Actual.

## Datos relacionados

- Se almacena en `gastos_variables.json` dentro de la carpeta del usuario.
- La información es privada por cuenta autenticada.
