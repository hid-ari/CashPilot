# Ahorros

Esta pantalla permite registrar y administrar los ahorros del usuario.

## Qué permite hacer

- Agregar filas.
- Editar categoría, descripción y monto ahorrado.
- Filtrar por categoría.
- Marcar filas para borrar.
- Exportar en CSV.

## Campos

- Categoría.
- Descripción.
- Ahorro.

## Datos relacionados

- Se almacena en `ahorros.json` dentro de la carpeta del usuario.
- Se utiliza para el resumen financiero y las gráficas.
