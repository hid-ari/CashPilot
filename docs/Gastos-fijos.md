# Gastos fijos

Esta pantalla permite administrar los gastos fijos del usuario.

## Qué permite hacer

- Agregar filas nuevas.
- Editar categoría, descripción, presupuesto y valor actual.
- Filtrar registros por categoría.
- Marcar filas para eliminarlas.
- Descargar los datos en CSV.

## Campos

- Categoría.
- Descripción opcional.
- Presupuesto.
- Actual.

## Datos relacionados

- Se guarda en `gastos_fijos.json` dentro de la carpeta del usuario.
- Cada usuario mantiene sus propios gastos fijos de forma independiente.
